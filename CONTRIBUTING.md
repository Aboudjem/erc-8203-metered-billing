# Contributing

This is a non-normative profile for [ERC-8203](https://github.com/ethereum/ERCs/pull/1614). Feedback, edge-case reports, and reference implementations are all welcome.

## Where to discuss

- ERC-8203 main thread on Ethereum Magicians: https://ethereum-magicians.org/t/erc-8203-agent-off-chain-conditional-settlement-extension-interface/28041
- Open a GitHub issue here for repo-level questions

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b notes/your-change`)
3. Edit `docs/profile.md` or add a worked sub-example under `docs/examples/`
4. Open a PR with a one-paragraph description of the change

## Style

- Match the existing voice: protocol-engineer talking to peers, not a standards committee
- Acknowledge prior art when extending or contradicting it
- Keep diffs scoped: one concern per PR
- No em-dashes, no marketing prose

## Reporting issues

Open an issue with:

- The section or example you're commenting on
- What you expected vs. what's there
- A concrete suggestion if you have one
